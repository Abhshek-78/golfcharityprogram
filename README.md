# DrawGolf Charity Platform

A React-based charity incentive web app that allows users to log golf scores, participate in a monthly draw, track charity contributions, and manage team/admin reports.

## Project Structure

- `index.html` - Vite entrypoint
- `package.json` - dependencies + scripts
- `vite.config.js` - Vite config for React
- `.gitignore` - ignored files
- `src/`
  - `main.jsx` - React app bootstrap
  - `App.jsx` - full app implementation (routing, components, logic)
  - `index.css` - base global styles

## Setup

1. `npm install`
2. `npm run dev`
3. Open the URL reported by Vite (typically `http://localhost:5173`)

## Build

- `npm run build`
- `npm run preview` for local production preview

## Notes

- The app uses client-side state and mock data; no backend is included.
